Detailed Description

This program is proof that the correct environment was set up to work with the ATMEGA328p.

Author
Pratik Kunkolienker
Date
21 Jan 2019

This program was made to run on the ATMEGA328p running at 16MHz. The program is set to blink the built-in LED connected to pin 13 at an interval of 1 second. This can be modified by changing the value of the t variable in the main function. In addition, if a LED is connected to pin 8 via a current limiting resistor (1k in my case) the LEDs will flash in an alternating manner.
